What is HTML?

HTML stands for Hypertext Markup Language. The purpose of this language is to determine the layout and content of your page while CSS is used to style it. Think of it as the skeleton of your page.


There are many elements that you can write in html, but keep in mind the purpose of these is to determine the layout, not style. For example, you wouldn't want to use the <h1> element for a title just because it has a big font size, rather you'd use it to mark where your title will go.

Getting Started

Let's start by declaring the document type with <!DOCTYPE html> . This lets the browser know what version of html you're running. HTML5 (current HTML) will always have the same tag as shown. Then we'll follow up with the <html> , <head> , and <body> elements. <html> contains all HTML elements, <head> contains data you wouldn't want to show on your main page such as font links, and <body> contains your main content. All put together it'll look like this:

  <!DOCTYPE html>
  <html>
    <head>
    </head>
    <body>
    </body>
  </html>

Remember to always use the closing tags (tag with slash) as shown above. There are some exceptions, these are known as self closing tags, but we'll get there later.

Headers

Header elements are used to mark where a title will go. <h1> is typically used as the main page title. There should only be one <h1> element. <h2> through <h6> are used in titles below the page title such as article headings. All introductory information such as titles or navbars will go in the <header> tags.


Lets use these headers in an example. First, we'll use the <h1> element for the page title, and then we'll use <h2> to title an article below.

  <!DOCTYPE html>
  <html>
    <head>
    </head>
    <body>
      <header>
        <h1> HTML Introduction </h1>
      </header>
    </body>
  </html>

Page Content

Now that we've finished laying out the formalities we can start adding content to the page. For now, I'll show you how to add text and images. For text that isn't a title of some sort, you'll want to use the paragraph element, written as <p>. For images, you'll use the self closing tag <img>.


The paragraph element is used in the same way as the header elements. However, <img> is different. To use it, you'll have to add a href attribute, aka a hypertext reference. It'll look like this: <img href="#">. The # is a placeholder, but that's where your link will go. If it's a photo on your computer, you'll want to add it to the same folder as your other webpage files. The folder with all your webpage content and files is known as the root folder. When you add it to the folder, you'll simply type the file name where the # is, including the file type at the end. For example, <img href="link to image"> or <img href="RootPic.png">.

Divs and CSS

After putting in your content, you'll probably be wondering about how to style it by changing fonts, sizes, etc. This is where CSS comes into play, but we'll talk about it a bit later. It's better to understand HTML before moving on to styling. However, I did want to give a little explanation of css and divs without diving into it.


Let's start with divs. In VERY simple terms, div's essentially create a specific area for all elements nested within. Nesting an element means putting an element in an element. For example, <div><p> Lorem Ipsum </p></div>. The large grey rectangles with code inside them that I wrote above are divs that I gave a height, width, and background color to. I gave it all those properties by using CSS.
