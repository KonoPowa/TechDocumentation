What is HTML?

HTML stands for Hypertext Markup Language. The purpose of this language is to determine the layout and content of your page while CSS is used to style it. Think of it as the skeleton of your page.


There are many elements that you can write in html, but keep in mind the purpose of these is to determine the layout, not style. For example, you wouldn't want to use the <h1> element for a title just because it has a big font size, rather you'd use it to mark where your title will go.
